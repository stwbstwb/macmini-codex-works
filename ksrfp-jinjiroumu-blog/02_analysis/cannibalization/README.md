# 02_analysis/cannibalization

過去記事との重複・カニバリ確認結果を置きます。

例:

- 既存記事テーマ分類
- 新規候補テーマとの類似度チェック
- 重複リスクのある既存URL一覧

主な出力:

- `posted_articles_inventory.csv`: 投稿済み記事の基本一覧
- `posted_articles_category_summary.csv`: カテゴリ別件数
- `posted_articles_theme_inventory.csv`: 投稿済み記事ごとの主要テーマ分類
- `posted_articles_theme_summary.csv`: 主要テーマ別件数
- `posted_articles_theme_report.md`: 主要テーマ分類の確認レポート
