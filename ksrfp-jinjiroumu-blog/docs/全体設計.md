# 全体設計

## 目的

柏谷横浜社労士事務所のコンテンツマーケティングとして、人事労務だよりを起点にSEO記事を作成し、WordPressへ投稿する月次自動化を構築する。

## 処理フロー

1. Googleドライブから最新の人事労務だよりPDFを取得する。
2. PDF本文を抽出し、ニュース・法改正・助成金・実務解説を小トピック単位に分割する。
3. 投稿済み記事一覧と照合し、重複・カニバリリスクを確認する。
4. GSC/GAデータを参照し、SEO需要や既存流入テーマとの相性を確認する。
5. 中小企業への実務影響、最新性、法改正の重要性を加味してテーマを選定する。
6. 既存プロンプトのルールに沿って記事構成と本文を生成する。
7. 法改正・制度情報は一次情報で確認する。
8. 確認者向けのテキストファイルを生成し、指定Google Driveフォルダへ保存する。
9. アイキャッチ画像を生成する。
10. WordPress REST APIで月曜日朝9:00の日付を設定した下書きを作成する。
11. 実行ログと処理済み状態を保存する。
12. 成功・失敗を問わずメール通知する。
