# 01_inputs

記事作成・分析・自動化に使う元データを保管するフォルダです。

## サブフォルダ

| フォルダ | 用途 |
| --- | --- |
| `gsc/` | Google Search ConsoleのエクスポートCSV |
| `ga/` | Google AnalyticsのエクスポートCSV |
| `posted-articles/` | 投稿済み記事一覧CSV |
| `prompts/` | 既存の手動作成プロンプト |
| `newsletters/local-samples/` | 手動で受け取った人事労務だよりPDFサンプル |
| `newsletters/drive-downloads/` | Googleドライブから自動取得したPDF |
